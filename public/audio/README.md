# 배경음악(BGM) 넣는 법

이 폴더에 `bgm.mp3` 파일을 넣으면 또또 월드의 배경음악으로 자동 재생됩니다.
(파일이 없으면 효과음만 나오고, 에러 없이 조용히 무시됩니다.)

## 저작권 안전한 CC0 / 로열티프리 음원 받는 곳

- **Pixabay Music** — https://pixabay.com/music/ (대부분 무료·출처표시 불필요)
- **Free Music Archive** — https://freemusicarchive.org/ (라이선스를 **CC0**로 필터)
- **Incompetech (Kevin MacLeod)** — https://incompetech.com/ (대부분 CC-BY, 출처표시 필요 — 안전하게 쓰려면 위 두 곳 권장)

> 💡 동화/산책 톤의 잔잔한 어쿠스틱·피아노·로파이 곡을 추천합니다.

## 적용

1. 위 사이트에서 마음에 드는 **CC0** 또는 로열티프리 mp3를 다운로드
2. 파일 이름을 `bgm.mp3` 로 바꿔 이 폴더(`public/audio/`)에 저장
3. 사이트를 새로고침 → 첫 클릭/터치 시 음악이 시작됩니다
4. 우상단 🔊 버튼으로 음소거를 켜고 끌 수 있습니다

효과음(점프·손·엎드려·데굴·앉아·빵야·뱅글)은 코드로 합성되어 별도 파일이 필요 없습니다.
